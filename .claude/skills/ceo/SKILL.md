# Hoạch Định Chiến Lược Cơ Bản

Skill hỗ trợ CEO/founder hoạch định chiến lược doanh nghiệp theo quy trình 6 bước.
Đây là bộ công cụ cơ bản để thực hành - cần bổ sung thêm framework và kiến thức chuyên ngành cho từng doanh nghiệp cụ thể.

## Quy trình 6 bước

| Bước | Nội dung | Công cụ phân tích |
|------|----------|-------------------|
| 1 | Giới thiệu doanh nghiệp | Thu thập thông tin từ user |
| 2 | Phân tích ngành | PESTEL, Porter's Five Forces, Value Chain |
| 3 | Phân tích đối thủ | So sánh trực tiếp, khoảng trống thị trường |
| 4 | Rà soát sản phẩm & SP mới | BCG Matrix |
| 5 | Lập chiến lược | SWOT/TOWS, STP, định vị, trụ cột chiến lược |
| 6 | Kế hoạch tài chính | Doanh thu, chi phí, KPI theo phân khúc |

## Quy trình làm việc

### Bước 1: Hiểu doanh nghiệp

Trước khi phân tích, cần hiểu rõ doanh nghiệp. Thu thập thông tin:

**Bắt buộc:**
- Công ty là gì, đang làm gì, lĩnh vực nào
- Sản phẩm/dịch vụ: tên, giá, kênh bán, đối tượng khách hàng
- Khách hàng: ai đang mua, tại sao chọn công ty thay vì đối thủ
- Báo cáo tài chính: tối thiểu năm gần nhất
- Đối thủ: tên và mô tả ngắn

**Có thì tốt:**
- Nhân sự, phòng ban, cấu trúc công ty
- Doanh thu, lợi nhuận từng sản phẩm
- Chiến lược hiện tại
- Vấn đề đang gặp phải

Nếu user upload file, đọc kỹ. Nếu user đưa link website, dùng web search để tìm hiểu.

### Bước 2: Phân tích ngành

Dùng web search để research, kết hợp kiến thức user bổ sung.

**PESTEL:**
- Political: chính sách, quy định ảnh hưởng ngành
- Economic: tăng trưởng GDP, lạm phát, sức mua
- Social: xu hướng tiêu dùng, nhân khẩu học
- Technological: công nghệ mới, chuyển đổi số
- Environmental: yêu cầu môi trường
- Legal: luật, thông tư liên quan

**Porter's Five Forces:**
- Đối thủ hiện tại: ai, mạnh yếu gì
- Đối thủ tiềm năng: rào cản gia nhập cao/thấp
- Sản phẩm thay thế: có gì thay thế được
- Quyền lực nhà cung cấp: phụ thuộc ai
- Quyền lực khách hàng: khách có nhiều lựa chọn không

**Value Chain:** Xác định vị trí công ty trong chuỗi giá trị ngành. Ai ở upstream, ai downstream. Giá trị tạo ra ở khâu nào.

Sau khi phân tích, luôn hỏi user: "Thông tin này có đúng với thực tế anh/chị đang thấy không? Có gì cần bổ sung hoặc điều chỉnh?"

### Bước 3: Phân tích đối thủ

Search web cho từng đối thủ. Tạo bảng so sánh:

| Tiêu chí | Công ty user | Đối thủ A | Đối thủ B |
|----------|-------------|-----------|-----------|
| Sản phẩm chính | | | |
| Giá | | | |
| Kênh bán | | | |
| Điểm mạnh | | | |
| Điểm yếu | | | |
| Phân khúc KH | | | |

Kết luận: khoảng trống thị trường, cơ hội khác biệt hoá.

Nhắc user: Claude chỉ phân tích được thông tin công khai. User biết nhiều hơn từ thực tế kinh doanh - hãy bổ sung.

### Bước 4: Rà soát sản phẩm (BCG Matrix)

Phân loại sản phẩm/dịch vụ:
- **Ngôi sao** (Star): tăng trưởng cao, thị phần cao → đầu tư mạnh
- **Bò sữa** (Cash Cow): tăng trưởng thấp, thị phần cao → thu hoạch
- **Dấu hỏi** (Question Mark): tăng trưởng cao, thị phần thấp → đầu tư hoặc cắt
- **Chó** (Dog): tăng trưởng thấp, thị phần thấp → cân nhắc cắt bỏ

Lưu ý quan trọng khi output:
- BCG Matrix chỉ là 1 công cụ, không đủ để quyết định. Cần kết hợp framework khác.
- Luôn hỏi CEO phản biện: Claude có thể đề xuất cắt sản phẩm A nhưng CEO biết nó đang giữ chân khách lớn.

### Bước 5: Lập chiến lược

**SWOT/TOWS:**
- S (Strengths) và W (Weaknesses): từ bước 1 (nội bộ)
- O (Opportunities) và T (Threats): từ bước 2-3 (bên ngoài)
- TOWS: chiến lược cho từng tổ hợp (SO tấn công, ST phòng thủ chủ động, WO khắc phục, WT phòng thủ)

**STP:**
- Segmentation: chia khách hàng thành các nhóm
- Targeting: chọn nhóm nào tập trung
- Positioning: công ty là gì trong tâm trí nhóm đó

**Chiến lược dài hạn / trung hạn / ngắn hạn:**
- Dài hạn: ngành sẽ thay đổi thế nào, công ty muốn ở vị trí nào
- Trung hạn: doanh thu, khách hàng, thị phần cần đạt, sản phẩm cần có
- Ngắn hạn (1 năm): 3-5 trụ cột chiến lược, mục tiêu theo quý, KPI, rủi ro

**Lợi thế cạnh tranh & rào cản:** Đánh bằng gì (giá, chất lượng, tốc độ, công nghệ, mạng lưới)? Giữ lợi thế bằng cách nào?

**Stress-test:** Sau khi có chiến lược, tự phản biện:
- Giả định nào quá lạc quan?
- Nếu đối thủ phản ứng mạnh thì sao?
- Kịch bản xấu nhất là gì?

### Bước 6: Kế hoạch tài chính

Lượng hoá chiến lược thành con số:
- Kế hoạch doanh thu: chia theo sản phẩm, kênh bán, theo quý
- Kế hoạch chi phí: nhân sự, marketing, vận hành, đầu tư mới
- Lợi nhuận dự kiến theo tháng và quý
- KPI theo dõi hàng tháng

Lưu ý: Claude tính toán dựa trên giả định. User phải kiểm tra giả định có thực tế không.

## Nguyên tắc quan trọng

1. **Dữ liệu thực > giả định:** Ưu tiên search web lấy số liệu thật. Khi phải giả định, ghi rõ "[Giả định]".

2. **CEO phản biện:** Luôn nhắc user kiểm tra và phản biện kết quả. Claude đề xuất, CEO quyết định.

3. **Mỗi nhóm khách hàng cần cách tiếp cận khác nhau:** Sản phẩm khác, giá khác, kênh bán khác.

4. **Xuất file:** KHÔNG tự động tạo file Word/Excel giữa chừng - tốn thời gian và dễ lỗi. Output kết quả trực tiếp trong chat. Chỉ tạo file khi user chủ động yêu cầu, hoặc khi đã hoàn thành xong một cụm phân tích lớn (ví dụ: xong bước 5 thì gộp tất cả thành 1 file chiến lược).

5. **Đây là bài tập thực hành:** Skill này cung cấp framework cơ bản. Chiến lược thực sự cần nhiều hơn - nhiều framework, nhiều dữ liệu, nhiều vòng phản biện.
